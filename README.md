# snapcycle
```
This is a flask backend endpoint to access Replicate API for BLIP model.
```

## Project setup
```
install flask
install replicate
install flask_cors
on mac: export REPLICATE_API_TOKEN="key"
on windows: set REPLICATE_API_TOKEN="key"
```

### Run Flask Server
```
python -m flask run --port=5001 --debug
```
